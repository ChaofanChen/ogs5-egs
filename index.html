OGS5-EGS

OGS5 branch for deep geothermal reservoir modeling

Download .zip Download .tar.gz View on GitHub

About

This website provides information about OGS5-EGS code. The code is a branch of OpenGeoSys5 (www.opengeosys.org) specially focusing on deep geothermal reservoir modeling with discrete fractures. The code is based on the OGS version 5.3.3 and is not fully compatible with the official version.
