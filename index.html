About

This website provides information about OGS5-EGS code. The code is a branch of OpenGeoSys (www.opengeosys.org) v5.3.3 for deep geothermal reservoir modeling developed by @norihiro-w.

Please be aware that

  • The code is not fully compatible with the official version.
  • Use of the official OGS version is recommended if your problem can work with it.

Features

  • Groundwater flow and heat transport modeling in porous media and discretely fractured porous media based on FEM
  • Unstructured mesh with mixed dimensional elements
  • Parallelization: OpenMP with Lis / MPI with PETSc

Third-party libraries

The code uses the following external libraries

  • ExprtTk
  • RapidXML
  • Lis (optional) *Latest Lis is not supported yet. please use older one.
  • PETSc (optional)
  • MKL (optional)

How to compile

with Lis

  1. Compile Lis
  2. Download source codes from above links and extract it
  3. Compile OGS as
cd [source-directory]
mkdir ../build
cd ../build
cmake ../[source-directory] -DOGS_FEM_LIS=ON -DLIS_INCLUDE_DIR=[lis include dir] -DLIS_INCLUDE_DIR=[lis include dir] -DLIS_LIBRARIES=[lis library path]
